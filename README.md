# Stock Market Analysis

## Overview
Analysis of historical stock price data for NVIDIA (NVDA), Apple (AAPL), and Microsoft (MSFT) using Microsoft Excel. The project covers 5 years of daily trading data (2019–2024).

## Tools Used
- Python (yfinance library) — data collection
- Microsoft Excel — data analysis and visualization

## What I Did
- Downloaded 5 years of daily closing prices for 3 stocks
- Calculated daily returns, average return, volatility, and median return
- Built price trend charts for each stock
- Compared all 3 stocks in a summary table

## Key Findings
| Stock | Avg Daily Return | Volatility |
|-------|-----------------|------------|
| NVDA  | 0.30%           | 3.30%      |
| AAPL  | 0.10%           | 1.90%      |
| MSFT  | 0.10%           | 1.80%      |

NVDA had the highest return but also the highest risk, demonstrating the classic risk-reward tradeoff in financial markets.

## Data Source
Yahoo Finance via yfinance Python library
